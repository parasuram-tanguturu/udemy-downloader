# Curriculum Creation Flow

This document explains how the `curriculum.md` file is generated from course data with detailed visual flow diagrams.

## Overview

The curriculum markdown file is a comprehensive, human-readable document that provides a rich overview of the course structure, including course metadata, instructor information, learning objectives, prerequisites, and a detailed chapter-by-chapter breakdown with lecture types and durations. It is generated **only when explicitly requested** using the `--curriculum-only` flag.

## Usage Examples

### Example 1: Basic Generation

```bash
python main.py \
  -c https://www.udemy.com/course/example \
  -b {bearer_token} \
  --curriculum-only
```

**Output:**
```
out_dir/example-course/curriculum.md
```

---

### Example 2: With Chapter Filter

```bash
python main.py \
  -c https://www.udemy.com/course/example \
  -b {bearer_token} \
  --curriculum-only \
  --chapter "1,3,5"
```

**Output:**
- Only chapters 1, 3, and 5 included in markdown

---

### Example 3: With Course ID as Name

```bash
python main.py \
  -c https://www.udemy.com/course/example \
  -b {bearer_token} \
  --curriculum-only \
  --id-as-course-name
```

**Output:**
```
out_dir/6195287/curriculum.md
```
